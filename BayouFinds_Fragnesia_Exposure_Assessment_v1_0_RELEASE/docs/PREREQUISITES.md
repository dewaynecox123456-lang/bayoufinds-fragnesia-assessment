# Prerequisites

Required:
- bash
- uname
- grep
- awk
- sed

Recommended:
- kmod tools (`modinfo`, `lsmod`)
- iproute2 (`ip`)
- rpm-ostree for Fedora Atomic / Silverblue systems

Permissions:
- Standard user execution is supported.
- Elevated permissions may provide additional system visibility depending on local policy.

The tool does not auto-install packages and does not auto-modify system configuration.
