# Quickstart

## Preferred Execution

```bash
chmod +x fragnesia-assessment.sh
./fragnesia-assessment.sh --quick
```

## Fallback Execution

If executable permissions were not preserved after ZIP extraction:

```bash
bash fragnesia-assessment.sh --quick
```

## Operator Acknowledgment

When prompted, type:

```text
I ACCEPT
```

## Output Locations

Reports are written to:

```text
./output/
```

Logs are written to:

```text
./logs/
```

## Important

Depending on your extraction method, browser, filesystem,
or desktop archive utility, executable permissions may
need to be manually restored.

This is normal Linux behavior for ZIP archives.
